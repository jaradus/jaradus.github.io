Colophon

Version controlled on and served from my Github profile, this site was built using jQuery and Backbone on a sixteen-column grid adapted from Skeleton.

It was designed in Sketch and written using Sublime Text 3. Photographs were taken on a Nikon D40 and processed in Aperture. Photoshop provided the graphics. Header fonts are Helvetica Neue, and the serif body text uses Athelas.
